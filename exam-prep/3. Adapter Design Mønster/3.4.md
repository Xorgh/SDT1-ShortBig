# Adapter-eksempel

## Hvad bliver adapteren koblet sammen med?

- `NotificationManager` som klientens forventede interface
- `CustomAlertBox` som den provided klasse med et andet interface
- `CustomAlertBoxAdapter` som oversætter mellem de to

## Talepunkter

- Forklar at klienten kun forventer `notify(message, type)`
- Peg på at `CustomAlertBox` i stedet tilbyder `showAlert(message, title, alertType)`
- Beskriv hvorfor `CustomAlertBoxAdapter` er det naturlige mellemled, fordi den kan mappe både type og titel
