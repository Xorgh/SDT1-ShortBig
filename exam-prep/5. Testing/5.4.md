# Teststruktur

## Hvordan er testene organiseret?

- Unit tests
- `test/business/services`
- Integration tests
- `test/integration`

## Talepunkter

- Peg på hvor unit tests og integrationstests ligger i projektet
- Nævn at `test/business/services` tester service-laget isoleret
- Nævn at `test/integration` tester samarbejde mellem flere rigtige komponenter
