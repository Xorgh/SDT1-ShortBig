# Exam Prep Workspace

This folder is for study artifacts and revision support, not application changes.

## Suggested workflow

Use Codex with prompts like:

1. "Build me a topic map for this repo using `documentation/` first."
2. "Summarize assignment 7 and the implementation files it depends on."
3. "Quiz me on the stock market simulation design, one question at a time."
4. "Turn my weak areas into a cram sheet in `exam-prep/weak-spots.md`."
5. "Compare the requirements with the current implementation and explain gaps without changing code."
